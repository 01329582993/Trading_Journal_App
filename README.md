# Trading Journal App

A powerful trading journal application built with React Native (Expo) and Supabase.

## Features
- Track trades with detailed metrics.
- Supabase integration for cloud storage.
- Real-time performance dashboard.
- User authentication.

## Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Run `npx expo start`.

## Supabase Setup
Ensure you have the following tables in your Supabase database:
- `trades`
- `targets`
- `trading_days`
- `performance_metrics`
